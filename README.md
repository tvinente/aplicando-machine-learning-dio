#  **Treinamento de um algoritmo de Inteligência Artificial**

## **Etapa 1: Criando um Dataset** 
### Para esse projeto foi criada uma base de dados (dataset) para o treinamento de um algoritmo baseado em Redes Neurais Artificiais. Para essa tarefa foram utilizados duas clases (panda_gigante e panda_vermelho): classes de objetos com 200 imagens para cada uma delas.
- As imagens dos Pandas Gigantes (*Ailuropoda melanoleuca*) foram obtidas no iPanda-50
- As imagens dos Pandas Vermelhos (*Ailurus fulgens*)  foram obtidas no Wikimedia Commons
- Imagens redimencionadas para 224x224 com fundo preto
- Imagens renomeadas mantendo padronização (img_001)
- Linguagem de Programação: Python
- Editor de Código: VSCode
